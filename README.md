# AutoCryptoBot 🔥

بوت تداول أوتوماتيكي ذكي لمنصة Binance (Spot و Futures)

### مميزات البوت
- استراتيجيات مدعومة بالذكاء الاصطناعي: DCA، Grid، Trend Following  
- تحليل السوق والمشاعر (Sentiment) باستخدام Ollama أو GPT محليًا  
- سهل التعديل والتشغيل على جهازك الخاص  
- دعم متعدد العملات والأزواج  

### خطوات التشغيل (بسيطة)
1. `git clone https://github.com/Ssemo/AutoCryptoBot.git`  
2. `cd AutoCryptoBot`  
3. `pip install -r requirements.txt`  
4. أنشئ ملف `.env` وضيف فيه:

5. **تحذير هام جدًا ⚠️**  
التداول الآلي في العملات الرقمية مخاطرة عالية جدًا!  
- قد تخسر كل رأس مالك أو جزء كبير منه.  
- البوت ليس مضمون الربح، والسوق متقلب.  
- **جرب دائمًا على حساب تجريبي (Binance Testnet) أولاً** لمدة أيام أو أسابيع قبل أي فلوس حقيقية.  
- لا تستخدم API keys بصلاحيات سحب (withdraw) أبدًا!  

المسؤولية عليك 100%، استخدمه على مسؤوليتك الخاصة.
